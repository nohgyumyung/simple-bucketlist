# Simple BucketList
간편하고 쉽게 버킷리스트를 관리할 수 있는 앱입니다.
기본적인 버킷리스트 추가, 수정 및 삭제 기능 등을 지원하고 있으며, 앱의 테마 색상 변경 및 다크 모드 기능을 지원합니다.
